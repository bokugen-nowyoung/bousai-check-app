## 1. 診断で聞いている10の質問

| # | 質問 | 選択肢 |
|---|------|--------|
| 1 | 住居について | 戸建て(旧耐震/新耐震)・マンション(低中層/高層)・賃貸 |
| 2 | 家族構成（複数選択） | 乳幼児・小中学生・高齢者・ペット・一人暮らし・夫婦 |

- 選択肢の粒度や言葉づかいが実態に合っているか

## 2. 質問の回答 → 表示される対策とその優先度

優先度は「高（スコア22点以上）」「中（13〜21点）」「低（12点以下）」の3段階で表示しています。
以下は対策ごとの「表示条件」と「優先度の付け方」です。

| 対策 | 表示される条件 | 優先度の付け方 |
|------|----------------|----------------|
| 備蓄を7日分まで拡充 | 備蓄が3日分以下／少量／なし | なし=30点、少量=24点、3日分=14点 |
| 非常持ち出し袋の点検・更新 | 古い／一部／なしのいずれか | なし=26点、一部=20点、古い=14点 |
| 子ども用持ち出し袋の準備 | 乳幼児or小中学生がいて、袋が不十分 | 一律18点 |
| 洪水対策：垂直避難の備蓄移動 | 洪水リスクが「高い」or「多少ある」 | 高い=26点、多少=12点 |
| 洪水対策：避難シミュレーション | 同上 | 高い=16点、多少=8点 |
| 情報収集手段の確保 | 常に表示（全員に推奨） | ハザードマップ未確認なら10点、確認済みなら6点 |
| 避難ルートの実地確認 | 避難場所が「把握のみ」or「未把握」 | 未把握=22点、把握のみ=10点 |
| 学校〜自宅の迎えルート確保 | 小中学生がいて、日中誰かが在宅でない | 一律16点 |
| 家族の連絡方法を決める | 連絡方法が「あいまい」or「未決定」 | 未決定=24点、あいまい=12点 |
| 車を使わない判断基準 | 車の備えが不足 かつ 洪水リスクあり | 一律14点 |
| 家具の固定 | 常に表示 | 旧耐震の戸建てなら18点、それ以外10点 |
| 窓ガラスの飛散防止フィルム | 常に表示 | 一律8点 |
| 高齢者の避難方法の具体化 | 高齢の家族がいる | 一律20点 |
| ペット用の備蓄・避難方法 | ペットがいる | 一律14点 |
| ハザードマップの確認 | 未確認／一部確認 | 未確認=20点、一部=8点 |

**確認したいこと**
- 点数配分（重み付け）が防災士の感覚とズレていないか
  （例：「ハザードマップ未確認」はもっと最優先にすべき、など）
- 「常に表示」にしている対策（情報収集手段・家具固定・窓ガラス）の扱いは適切か
- 表示条件の組み合わせ方（AND/OR）に違和感がないか
